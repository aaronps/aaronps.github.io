== Google Chrome

Start with proxy::
    `chrome --proxy-server=socks5://127.0.0.1:1080 --host-resolver-rules="MAP * ~NOTFOUND, EXCLUDE 127.0.0.1"`

See current proxy config status::
    link:chrome://net-internals/#proxy[]
