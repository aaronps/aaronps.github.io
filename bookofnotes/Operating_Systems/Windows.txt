====== Windows ======

===== CMD =====

{{{code: lang="dosbatch" linenumbers="True"
rem Disconnect network drive
net use drive: /delete

rem Symlinks
rem file link:
mklink filename path/to/link/target

rem softlink:
mklink /D dirname path/to/link/target

rem hardlink:
mklink /H filename path/to/link/target

rem junction:
mklink /J filename path/to/link/target


rem `echo` multiple lines and pipe output
C:\> (echo line one & echo line two) | otherprogram
}}}


===== VIM =====

Set default language to english and utf8: write on ~/.vimrc

{{{code: lang="dosbatch" linenumbers="True"
set encoding=utf-8
set langmenu=en_US.UTF-8
language english
}}}
